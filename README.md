# act-lightcurve-viewer

Tools for analyzing and plotting ACT depth1 thumbnails and associated lightcurves.

To run, there is a script called `script.py` which can take arguments pointing to data files.

For example,

`python script.py -t [/path/to/thumbnails/] -c [coadd days]`

which will plot the thumbnails and lightcurves with the same number of coadded days.
